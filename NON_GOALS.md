# Non-Goals

MCP Scope explicitly does not aim to be:

- A generic MCP framework.
- A generic agent framework.
- A chatbot.
- A RAG chatbot.
- A SaaS dashboard.
- A login system.
- A billing system.
- A database-backed platform by default.
- A tool that calls external APIs in core scanning.
- An exploit generator.
- A tool for automatic attack execution.
- A product claiming complete security protection.
- A production-ready security guarantee.

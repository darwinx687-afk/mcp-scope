# Browser Posting Rules

These rules apply to any future manual or browser-assisted posting.

- All publishing remains manual.
- Chinese platforms must use Chinese copy and zh-CN image assets.
- Overseas platforms must use English copy and en image assets.
- Only use specified local SVG paths from `launch/assets/`.
- Do not select personal photos.
- Do not select random album images.
- Do not use external images or remote image URLs.
- V2EX must not be posted.
- Hacker News is text-only unless a human explicitly decides otherwise.
- Reddit and Hacker News require explicit human approval before posting if browser automation is used later.
- Pause for manual login.
- Pause for captcha, 2FA, or security checks.
- Do not save passwords.
- Do not bypass platform rules.
- Preview the post before publishing.
- Record the post URL in `launch/POSTING_TRACKER.md`.
- Record failures honestly.
- Do not spam retry after a failure.
- Do not publish the GitHub release from browser automation unless explicitly approved.

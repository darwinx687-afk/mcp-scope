# v0.2.0-preview Update Posting Checklist

Status: draft workflow. Do not publish automatically.

## Before Any Post

- Confirm `v0.2.0-preview` release notes are still draft unless a human explicitly publishes the release.
- Confirm no tag has been created automatically.
- Confirm no npm package has been published.
- Confirm no GitHub Marketplace listing has been published.
- Confirm copy does not claim official integration or a security guarantee.
- Confirm screenshots or images, if used, come from approved repository assets only.

## LinkedIn

- Use `linkedin.md`.
- English only.
- Preview before publishing.
- Confirm the GitHub repo link is present.
- Record the final URL in `UPDATE_POSTING_TRACKER.md` after manual publication.

## Xiaohongshu

- Use `xiaohongshu.md`.
- Chinese only.
- Preview before publishing.
- Confirm banned Chinese phrases are absent.
- Record the final URL in `UPDATE_POSTING_TRACKER.md` after manual publication.

## Stop Rules

- Stop for login, captcha, 2FA, or security checks.
- Stop if the platform changes the post text in a way that creates unsafe claims.
- Stop if the release has not been reviewed by a human.

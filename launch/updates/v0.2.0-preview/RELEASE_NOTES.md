# v0.2.0-preview

Status: public prerelease.

Theme: bilingual onboarding and limitation reduction.

## What Changed

- Added a visible README language switch near the first screen.
- Added repository metadata recommendations for the GitHub description and topics.
- Added limitations docs that separate current boundaries, v0.2 improvements, future versions, and non-goals.
- Added a tool metadata export guide for supported local JSON shapes.
- Improved discovery next-step hints so parsed candidates show a safe scan command.
- Prepared repeatable LinkedIn and Xiaohongshu update-post workflow files.

## Why It Matters

v0.1 made MCP Scope publicly reviewable. v0.2 makes the project easier to understand before adding larger features: clearer bilingual entry, clearer limitations, safer tool metadata guidance, and more obvious next steps after discovery.

## What Still Does Not Change

- No MCP server execution.
- No live `tools/list` requests.
- No external AI API calls.
- No npm package claim.
- No GitHub Marketplace claim.
- No security guarantee claim.
- Findings remain static risk signals, not proof of compromise or proof of safety.

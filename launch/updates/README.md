# Update Workflow

This folder contains versioned update-post materials for MCP Scope. These files are drafts and checklists only. They do not publish a release, create a tag, post to social platforms, publish npm, or publish GitHub Marketplace.

## Available Packs

- [v0.2.0-preview](v0.2.0-preview/): bilingual onboarding and limitation reduction update pack.
- [v0.3.0-preview](v0.3.0-preview/): draft SARIF and one-command audit mode update pack.

## Rules

- Review release notes before any publication.
- Preview social posts before publishing.
- Record final URLs in the versioned posting tracker after manual publication.
- Keep npm, Marketplace, official integration, and security guarantee claims out until those states are real.

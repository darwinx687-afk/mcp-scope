# 浏览器发布规则

这些规则适用于未来任何手动或浏览器辅助发布。

- 只使用 `launch/assets/` 中指定的图片路径。
- 不选择个人照片。
- 不选择随机相册图片。
- 遇到登录时暂停，交给人手动处理。
- 遇到 captcha、2FA 或安全检查时暂停。
- 不保存密码。
- 不绕过平台规则。
- 发布前先预览。
- 没有明确确认时，不自动发布 Reddit 或 Hacker News。
- 把发布链接记录到 `launch/POSTING_TRACKER.md`。
- 失败要如实记录。
- 失败后不要反复刷屏重试。
- 没有明确确认时，不通过浏览器自动把 GitHub release 转为正式发布。

# Screenshot Guide

Screenshots are optional for Phase 8. SVG assets are enough for launch packaging, and no PNG screenshots are generated in this loop.

## Safe Sources

- `examples/viewer/sample-combined-viewer.html`
- `examples/viewer/sample-combined-viewer.zh-CN.html`
- Local CLI output from safe example files.
- GitHub Action job summary from safe example files.

## Suggested Screenshots

- README first screen with the banner visible.
- CLI command output for `discover`.
- HTML viewer summary section.
- Diff report summary.
- GitHub Action summary table.

## Safety Rules

- No personal photos.
- No random album images.
- No real secrets.
- No private configs.
- No private paths, usernames, internal hostnames, or customer names.
- No screenshots that imply official integration or production-grade protection.

Review every screenshot before posting it publicly. Reports and summaries may contain tool names, descriptions, path hints, URL hosts, and config field names.

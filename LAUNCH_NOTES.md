# Launch Notes

MCP Scope is not launched yet.

Phase 5 adds stable JSON transparency reports, bilingual Markdown reports, self-contained local HTML viewer output, a local composite GitHub Action quality gate, report docs, and curated sample reports.

Do not describe this project as production-ready. Do not claim that it detects all malicious MCP servers or guarantees safety. The scanner and action do not execute MCP servers, send live `tools/list` requests, start a report web server, upload files automatically, or call external APIs.
